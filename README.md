Proyecto de Data Wrangling

En este proyecto se realiza la limpieza, transformación y preparación
de un dataset de e-commerce utilizando la librería Pandas en Python.

Tareas realizadas:
- Carga de datos desde CSV
- Exploración inicial del dataset
- Eliminación de registros duplicados
- Análisis de valores nulos
- Limpieza de columnas de texto
- Preparación del dataset para análisis posterior

El objetivo es aplicar técnicas de data wrangling
según lo visto en clase.
